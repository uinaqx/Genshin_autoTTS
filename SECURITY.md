# Security policy

Please report security issues privately to the repository owner instead of
opening a public issue containing credentials or exploit details.

Genshin_autoTTS never uploads captured screen images. Recognized dialogue text is
sent to Microsoft's Edge speech service to generate the selected neural voice;
the program does not silently fall back to a traditional system voice. Never include account
identifiers, screenshots, access tokens, generated voice caches, or proprietary
game data in bug reports without reviewing and redacting them first.
