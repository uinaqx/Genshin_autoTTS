# Genshin_autoTTS 快速开始

## 1. 启动

- 安装版：从开始菜单或桌面快捷方式启动 `Genshin_autoTTS`。
- 便携版：解压整个 ZIP 后运行 `Genshin_autoTTS.exe`，不要只复制单个 EXE。

首次启动会在 `%LOCALAPPDATA%\GenshinAutoTTS` 创建配置和缓存目录。

## 2. 设置游戏

1. 将游戏设为“窗口化”或“无边框窗口”。
2. 打开一段无配音对话并停留在完整字幕画面。
3. 在软件中先点击“全屏 16:9 预设”；如果识别不准，再分别框选角色名与字幕区域。
4. 点击“截屏识别测试”，确认日志中能看到角色和完整台词。

软件只读取屏幕像素，不注入进程、不读取游戏内存，也不修改游戏文件。

## 3. 选择录音包

- 留空“录音包清单”时使用内置的合法真人诊断样本，它只能验证播放链路，不包含游戏剧情配音。
- 使用自有或获授权的录音包时，选择其中的 `manifest.json`。
- 远程录音只在台词首次命中时按需下载，之后使用本地缓存。

## 4. 开始运行

点击“开始自动配音”，切回游戏。程序会等待字幕稳定，匹配同角色的真人录音并播放。没有安全匹配时会拒绝播放，不会自动用机器语音替代。

## 5. 常见问题

### OCR 无法识别

- 确认画面缩放与分辨率没有在框选后改变。
- 缩小框选范围，避开头像、任务提示和交互按钮。
- 使用“本地真实截图批量诊断”检查静态截图。

### 有识别结果但没有声音

- 先运行内置真人诊断样本，确认系统音量和播放设备正常。
- 检查录音包是否包含该角色与台词，以及 SHA-256 是否正确。
- 查看界面日志中的“缺少真人录音”或“完整性校验失败”提示。

### Windows 显示“未知发布者”

当前社区版本未购买代码签名证书。请只从本仓库的 GitHub Release 下载，并用同一 Release 中的 `SHA256SUMS.txt` 核对文件。不要从第三方网盘下载或关闭系统安全功能。
