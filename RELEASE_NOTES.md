# Genshin_autoTTS v0.3.0

首个面向普通 Windows 用户提供完整安装包的 Beta 版本。

## 下载选择

- `Genshin_autoTTS-Setup-x64.exe`：推荐，当前用户安装，无需管理员权限。
- `Genshin_autoTTS-Portable-x64.zip`：免安装，解压整个目录后运行。
- `SHA256SUMS.txt`：安装包和便携包的 SHA-256 摘要。

## 主要内容

- 正式版深色桌面界面。
- 本地 RapidOCR 与中文 ONNX 模型，无需另行安装 Python。
- 标准底部、上移/选项和超宽画面对话布局识别。
- 截屏识别测试与本地真实截图批量诊断。
- 贴近游戏字幕排版的模拟剧情窗口。
- 真人录音精确匹配与保守 OCR 漏字纠正。
- SHA-256 校验、按需下载和容量受限缓存。
- 内置合法真人诊断样本、中文快速指南和示例配置。

## 使用前请注意

- 本项目不附带游戏截图、台词库、官方音频或未经授权的角色音色。
- 完整剧情覆盖需要用户自行选择合法的真人录音包。
- 当前安装包未使用商业代码签名证书，Windows 可能显示“未知发布者”。请只从本仓库 Release 下载并核对 SHA-256。
- 本项目是外部屏幕识别工具，不注入游戏进程、不读取游戏内存、不修改游戏文件。
